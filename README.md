# 사하의집 변비 관리 체크리스트

사하의집 이용인 변비 관리 체크리스트 단일 HTML 파일입니다.

## 파일

- `index.html`: 체크리스트 웹페이지

## GitHub Pages 사용 방법

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 `index.html`, `README.md`, `.nojekyll` 파일을 올립니다.
3. 저장소의 `Settings`로 이동합니다.
4. 왼쪽 메뉴에서 `Pages`를 누릅니다.
5. `Deploy from a branch`를 선택합니다.
6. Branch는 `main`, Folder는 `/root`를 선택합니다.
7. `Save`를 누릅니다.

잠시 기다리면 GitHub Pages 주소가 만들어집니다.
